Quickstart

The fastest way to run SFSVC is via the official Docker image. Once you have a valid license key from your license dashboard, you can be processing video in under 2 minutes.

Pull and run

# Pull the latest image
$ docker pull aurasense/sfsvc:latest

# Run with your license key (replace YOUR_KEY)
$ docker run --rm -it \
  -e SFSVC_LICENSE_KEY=YOUR_KEY \
  -p 8000:8000 \
  aurasense/sfsvc:latest
# Show all options
$ docker run --rm -it aurasense/sfsvc:latest --help
Your license key is available on the license dashboard after subscribing. The key is tied to your account; do not commit it to version control.

API Reference

The SFSVC container exposes a REST API on http://localhost:8000. All endpoints accept and return application/json.

The API is designed for on-device loopback calls from your application. Do not expose port 8000 directly to the internet without a reverse proxy and authentication.

Facade Mode API v1

New integrations should use the canonical Facade Mode API v1 contract. This contract includes vertical crack summary outputs and Golden Frame memory fields (has_baseline, area_growth_rate, risk_acceleration, and new_degradation_tiles).

Canonical contract: Facade Mode API v1

POST /v1/frames (Legacy)

POST /v1/frames
Legacy endpoint. Submit a raw frame (or pre-encoded spike event list) for neuromorphic processing. Returns immediately; results are available via /v1/events. New integrations should use Facade Mode API v1.

Request body

{
  "frame_id":   "f_1234",          // caller-assigned identifier
  "timestamp":  1741183200000,     // Unix ms
  "encoding":   "raw_bgr",         // "raw_bgr" | "spike_events"
  "width":      1280,
  "height":     720,
  "data_b64":   "..."              // base64-encoded frame bytes
}

Response

{
  "accepted": true,
  "frame_id": "f_1234",
  "queue_depth": 0
}

GET /v1/events (Legacy)

GET /v1/events
Legacy endpoint. Poll for detection events. Returns crack events emitted since the last poll (or since since timestamp if provided). New integrations should use Facade Mode API v1.

Query parameters

ParameterTypeDescription
sinceintegerUnix ms. Return events after this timestamp.
min_confidencefloat [0,1]Filter events below this confidence score.
limitintegerMax events per response. Default 100.

Response

{
  "events": [
    {
      "event_id":    "ev_9af3",
      "frame_id":    "f_1234",
      "timestamp":   1741183200043,
      "type":        "crack",
      "confidence":  0.943,
      "tile_x":      82,           // tile column [0, 159]
      "tile_y":      44,           // tile row    [0, 89]
      "lane_mask":   6,            // bitmask of firing lanes
      "bbox_px":     [656, 352, 672, 360]
    }
  ],
  "count": 1,
  "next_since": 1741183200044
}

Risk & Responsibility

SFSVC is decision-support software for inspection teams. It helps prioritize review, trend facade degradation, and surface risk indicators.

SFSVC output is not a structural certification and is not a substitute for legal/compliance review. Final structural and safety decisions must be made by qualified engineers under local codes and standards.

Q: Can I use SFSVC output as the sole basis for safety decisions?

A: No. SFSVC is designed to assist qualified professionals, not replace them.

Roadmap

  • Shear-angle risk channel (Sobel/atan2/sin(2theta)-based), to be exposed as max_shear_risk and per-tile shear factors in a future API revision.
  • Additional facade trend signals for long-horizon maintenance planning.

Shear-specific metrics are experimental and not yet part of the public API.

Licensing

SFSVC uses Ed25519 offline license keys. The license is validated locally on startup and via a periodic heartbeat call to https://www.aurasensehk.com/license/validate. The drone does not send video or spike data off-device during validation.

Your license key, active plan details, and Docker image download link are always available on your license dashboard.
