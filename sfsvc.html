SFSVC under NEPA — Facade Vertical

SFSVC is the neuromorphic perception core that powers AuraSense's facade inspection vertical. It converts UAV video into sparse spike representations, detects cracks and surface anomalies deterministically on the edge, and emits tamper-evident .cpse audit logs for every inspection session.

Core Capabilities

Air-Gapped Operation

Technical Specification

SFSVC Codec & NEPA Control Stack

Minimum hardware requirements, software dependencies, and runtime characteristics for SFSVC under the NEPA edge stack.

CPU Architecture x86_64 with AVX2 support (standard on Intel Haswell+ / AMD Ryzen). ARM64 supported with NEON fallback.
Operating System Ubuntu 22.04 LTS (primary). Ubuntu 20.04 LTS supported. Other Linux distros on request.
Storage SATA or NVMe SSD with sustained ≥ 50 MB/s sequential write throughput. Minimum 64 GB free for cpse log retention.
GPU Requirement None. SFSVC codec path is entirely CPU-based. AVX2 SIMD paths provide sufficient throughput for 1080p 30 fps UAV feeds.
Network (Runtime) Not required during inspection flights. Internet needed only for: (1) initial license activation, (2) signed Rule Bundle updates.
Encoding Latency Avg 2.50 ms, P99 ≤ 2.81 ms per frame at 1080p 30 fps on reference hardware (Intel Core i7-12700, 32 GB DDR5).
Compression Ratio 9.97× vs raw H.264 stream. 89.97% bandwidth reduction at 1080p 30 fps continuous operation.
Audit Log Format .cpse — hash-chained, Ed25519-signed binary log. Verified offline via cpse_replay CLI utility.
Detection Control Loop NEPA Lane 1 deterministic control at ~106 Hz. 6 ms safety contract. 0% SLA violations on benchmark dataset.
Rule Bundle Signing Ed25519 keypair. Public keys shipped in the device licence. Custom Rule Authoring available on Enterprise tier.
SDK Language C++ (private). SIMD AVX2 optimised with lock-free queues for Lane 1/2 separation. Python bindings available on Professional+.
Deployment Model Edge-native Linux process. Integrates with DJI / ArduPilot telemetry buses. Optional REST status endpoint for fleet dashboards.

Note on RSE sign-off: SFSVC under NEPA automates defect finding and recording with deterministic edge logic and cryptographic audit trails. A qualified Registered Structural Engineer (RSE) or equivalent professional must still review outputs and sign off on any statutory submission.
