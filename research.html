Research & Demo

Public reference implementations and experimental research.

NEPA Public Demo

NEPA is a public reference implementation.

NEPA (Neuromorphic Event Processing Architecture) demonstrates neuromorphic codec concepts in Python. It is provided as an educational and prototyping tool.

Important: NEPA is NOT the commercial SDK

  • NEPA is a Python-based research demo (~8 ms/frame)
  • SFSVC is the production C++ SDK (2.50 ms avg, 2.81 ms P99)
  • NEPA demonstrates concepts; SFSVC delivers production performance
What It Demonstrates

NEPA illustrates core neuromorphic principles:

Spike-based encoding

Frame-delta conversion to ON/OFF events

Temporal contrast detection

Edge detection through pixel change tracking

Data compression

Sparse representation reducing bandwidth

Crack detection pipeline

Basic defect identification workflow

These concepts are implemented in optimized C++ in the production SFSVC SDK with SIMD acceleration and deterministic scheduling.
